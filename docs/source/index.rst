Welcome to Bio-Volumentations' documentation!
==============================================

`Bio-Volumentations` is an image augmentation and preprocessing package for 3D, 4D, and 5D biomedical images.

The library offers a wide range of efficiently implemented transforms for volumetric images, an easy-to-use API,
and a neat Getting Started guide. On top of that, it can be used to manipulate your image data in any context
and with any major deep learning library.
