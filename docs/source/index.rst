Welcome to Bio-Volumentations' documentation!
==============================================

`Bio-Volumentations` is an image augmentation and preprocessing package for 3D, 4D, and 5D biomedical images.

The library offers a wide range of efficiently implemented transforms for multidimensional images with annotations, an easy-to-use API,
and this neat Getting Started guide. The library can be used to manipulate your image data in any context
and with any major deep learning library.
